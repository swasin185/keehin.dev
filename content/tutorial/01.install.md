# Nuxt 3 Installation
```bash
# nuxtjs init project
npx nuxi@latest init keehin.dev

# primevue
npm install nuxt-primevue primevue primeflex primeicons --save-dev

# nuxt-auth-utils
npx nuxi@latest module add auth-utils

# nuxt/content
npx nuxi@latest module add content
```
